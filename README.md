# 🎬 CGV Movie Web Project

> 映画情報の表示・予約を再現したWebサービス（個人プロジェクト）

---

## 📌 概要
本プロジェクトは、韓国の映画館サービスを参考にして制作した  
**映画情報閲覧・予約システムのWebアプリケーション**です。

映画一覧、詳細情報、検索機能などを実装し、  
実際のサービスに近い構成を目指しました。

また、一部機能では**Webクローリング技術**を活用しています。

---

## 🚀 主な機能
- 🎬 映画一覧表示（現在上映中・人気順など）  
- 🔍 映画検索機能  
- 📄 映画詳細ページ（タイトル・ジャンル・上映時間など）  
- 🕒 上映スケジュール表示  
- 🧾 予約機能（簡易実装）  
- 🌐 Webクローリングによるデータ取得  

---
