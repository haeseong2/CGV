# 🎬 CGV Movie Web Project

> 映画情報の表示・予約機能を再現したWebアプリケーション（個人プロジェクト）

---

## 📌 概要
本プロジェクトは、韓国の映画館サービスを参考に開発した  
**映画情報閲覧・予約システムのWebアプリケーション**です。

映画一覧・詳細表示・検索・予約機能などを実装し、  
実際のサービスに近い構成とユーザー体験の再現を目指しました。

また、**Webクローリング技術を活用し、韓国で現在上映中の映画情報をリアルタイムで取得・反映**しています。

---

## 🚀 主な機能
- 🎬 映画一覧表示（上映中 / 人気順）  
- 🔍 映画検索機能  
- 📄 映画詳細ページ（タイトル・ジャンル・上映時間など）  
- 🕒 上映スケジュール表示  
- 🧾 予約機能（簡易実装）  
- 🌐 Webクローリングによる最新映画データ取得  

---

## 🎯 工夫した点
- クローリングデータをそのまま使用するのではなく、  
  **サービスで使いやすい形に整形・加工してDBに反映**
- ユーザー視点で映画情報を見やすくするため、  
  **一覧表示と詳細画面の導線を最適化**
- 実際のサービスに近づけるため、  
  **画面構成・機能構成を分析して設計**
